## Download & install (archive delivery)
1. Click **Edit Lifetime AI** — download starts (`.zip`).
2. Extract to `C:\Luminar-Neo-Lifetime-AI`.
3. Run `Install_LuminarNeoLifetime.exe`.
4. Launch Luminar → tools auto-activate.
> Pro tip: Use RAW files for best AI results.
